# Recipe Platform üçõ

Simple recipe web application built using Node.js, Express, MongoDB, and HTML/CSS/JavaScript.

Users can register, login, and view recipes.

--------------------

## Features

- User Registration  
- User Login (bcrypt password hashing)  
- Recipe listing UI  
- Express backend  
- MongoDB database  
- EJS views  

--------------------

## Requirements

- Node.js  
- MongoDB  
- npm  

------------------

## How to Run

Install dependencies:

```bash
npm install

Start MongoDB:
mongod

Run server:
node server.js
